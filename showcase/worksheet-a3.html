🔍 Project Overview

A pocket-sized, open-source speech translation device built around the ESP32 microcontroller. It captures spoken audio, processes it through a 3-stage AI pipeline (Speech-to-Text → Translation → Text-to-Speech), and delivers translated audio through Bluetooth earbuds — enabling real-time, hands-free multilingual communication.

₹500 vs ₹10,800+
95% CHEAPER
🛡 Hallucination Rejection — 98%

Multi-layer filtering: VAD pre-filtering, confidence thresholding, no-speech probability check, segment length filtering, and a curated blacklist of 30+ known Whisper hallucinations.

🏗 System Architecture
📟

ESP32 Device

3 GPIO buttons (Record, Replay, Previous) + LED status indicator. Acts as BLE GATT Server advertising as "TranslatorBtn".

AI Processing Server

⚡ ESP32 GPIO Wiring
GPIO 4 → 🔴 Record Button
GPIO 5 → 🔁 Replay Button
GPIO 16 → ⏮ Previous Button
GPIO 2 → 💡 Status LED

All connections are soldered • Buttons use INPUT_PULLUP • Battery powered via power bank module

🚀 How to Use
1
Flash ESP32 firmware via Arduino IDE
2
Start AI Server — python server.py
3
Pair BT earbuds to phone, open Chrome PWA
4
Press & hold record button → Speak → Release
5
Hear translation through earbuds in ~3.8 seconds! 🎧
